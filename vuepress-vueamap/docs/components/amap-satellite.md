# AmapSatellite  卫星图层
基础用法：
::: demo
```vue
<div class="map-container">
  <amap-map>
    <amap-satellite></amap-satellite>
  </amap-map>
</div>
<style>
.map-container {
  height: 500px;
  width: 100%
}
</style>
```
:::
| 参数      | 类型      | 说明                       |
| :-------- | --------- | -------------------------- |
| `visible` | `Boolean` | 图层是否显示  默认：`true` |
