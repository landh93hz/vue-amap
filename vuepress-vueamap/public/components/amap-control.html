在使用插件的时候，要在项目初始化的时，在 options中的 plugins里引入所需插件

WARNING

使用插件之前一定要初始化，否则会报错！

import AmapVue from 'amap';
const options = {
  key: '0b8eff7c3cebc996161cf56812c23528',
  v: '1.4.15',
  plugins: [
    'AMap.Scale',
    'OverView',
    'MapType',
    'ToolBar',
    'ControlBar'
  ]
};
Vue.use(AmapVue, options);
